This app was inspired by my previous job at habrahabr.ru, geektimes.ru and mediaport.info. It is like CMS, but with support of collaborative editing.

Plans for the next version

  • Login with Google, Facebook, etc;
  • Import and export of story contents;
  • User profiles
  • Comments and story content annotations;
  • Integrated chat;
  • 'Libraries' for permanent storage of things like styleguide, educational reading for authors, editorial contact lists and bookmarks;
  • More granular access control with some new roles: 'observer', 'admin', 'trainee';
  • Story types (or genres) with enforced structure and size limitations.
  • Statistics module

Development

Source code of this app is available at Bitbucket: web client and server.
